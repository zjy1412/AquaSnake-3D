# AquaSnake 3D - 水族箱贪吃蛇

## 项目简介
这是一个基于OpenGL实现的3D贪吃蛇游戏，将经典的贪吃蛇游戏搬入了水族箱的3D环境中。

## 主要特性

### 游戏玩法
- 3D环境下的自由移动：使用WASD控制蛇在水族箱中进行三维运动
- 食物系统：收集食物使蛇身增长
- 障碍物系统：需要灵活操控避开各种障碍物

### 音效系统
- 动态背景音乐：菜单和游戏场景使用不同的背景音乐

### 用户界面
- 游戏菜单
- 实时分数显示
- 暂停/继续功能

## 操作说明
- W：向上转
- S：向下转
- A：向左转
- D：向右转
- R：重新游戏
